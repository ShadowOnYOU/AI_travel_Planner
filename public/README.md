# Public Assets Directory

This directory contains static assets that are served directly by Next.js.

Files in this directory are available at the root of your domain:
- `public/favicon.ico` → `/favicon.ico`
- `public/logo.png` → `/logo.png`

## Usage

Place your static assets here:
- Images
- Fonts  
- Icons
- Manifest files
- Robots.txt
- Sitemap.xml

These files will be served as-is by Next.js without any processing.
